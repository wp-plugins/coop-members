== Description ==

쿱 멤버스는 워드프레스협동조합에서 제공하는 한국형 소셜로그인 / 회원가입 플러그인 입니다.


* 저희 워드프레스협동조합은 공유경제의 힘을 믿습니다.
* 워드프레스의 정신 또한 공유경제의 힘으로 발전했습니다.
* 워드프레스와 협동의 힘으로 모두가 상생하는 IT생태계로 성장하길 바랍니다.
* 이제 시작이라 부족한 점이 많습니다.
* 앞으로 계속 다양한 플러그인을 출시하고 여러분과 함께 개발해서 널리 공유하겠습니다.
* 대한민국 워드프레스 활성화를 위해 많은 응원 부탁드립니다.

= 버전 0.1.1 (매뉴얼 준비 중) =
* 소셜로그인 지원: 페이스북, 트위터, 구글, 네이버, 카카오
* 회원가입 요소: 아이디, 이메일, 비밀번호, 이름, 유선전화, 휴대폰, 주소 등


* 공식홈페이지 http://www.wpcoop.kr
* 페이스북 공식 페이지 https://www.facebook.com/wordpresscoop

== Installation ==

1. 워드프레스 공식 홈페이지에서 coop-members 를 다운로드 받습니다.
2. 대시보드에서 설치를 진행하시려면 플러그인검색 에서 coop-members를 검색 하시면 됩니다.
3. 파일로 설치 하실 경우 wp-admin/plugins/coop-members 폴더를 생성하신 후 압축을 풀어 주시면 됩니다.
